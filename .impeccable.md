# DATUM — Impeccable Design Context

Inherits the SANO/WHAstudio visual system documented in `SANO_Brand_Graphic_Standard.md`. This file adds DATUM-specific context the standard doesn't cover.

## Design Context

### Users
**Primary**: Wilson (Principal of WHAstudio, MArch from Harvard GSD; bottleneck for design + ops decisions across ~50 active projects). **Secondary**: Carissa (interior design lead), PICs (project-specific points of contact), site supervisors. Team is 70+ in Surabaya, Indonesia. **Context**: used daily across desktop browsers and (eventually) mobile; field staff log progress from sites with patchy connectivity; PICs translate WhatsApp client conversations into structured DATUM entries; principals review pending decisions in 5–10 minute bursts between meetings. **Bahasa Indonesia is the operating language** — every UI string is Bahasa-first.

### Use Cases
- Replace scattered Trello + WhatsApp + email + Synology-NAS-files with one queryable surface
- Capture decisions, drawings, vendor quotes, materials, defects, photos, client requests, and progress on long-lived "subject" cards organized by drawing code (A01-03 DTP, A04 Tangga, A05 Kusen, A09 Detail Kamar Mandi, U02 Listrik, etc.)
- Ask the assistant questions in Bahasa with cited card references
- Capture via chat: type or drop a file → AI proposes target card + event kind → user confirms (high-risk routes through a review queue)
- Browse a Wilson morning brief, an activity feed, search, and a gate-readiness schedule view

### Brand Personality
**Earthen · tactile · documentary · disciplined · low-stimulus · trustworthy.** A well-made field notebook; a quantity surveyor's spec sheet; a folder of plot-drawing print-outs marked up at the site. Quiet confidence, no decoration for its own sake, truth-on-the-page. The opposite of cold-blue SaaS or glossy/neon developer tools.

### Aesthetic Direction
- **Palette**: oat / sand / warm near-black / warm near-white (see SANO standard §2). Every neutral is tinted warm — pure `#000` / `#FFF` / cold gray are forbidden.
- **Type**: Single family Space Grotesk at multiple weights (300/400/500/600/700). Card titles are 13px **bold uppercase +0.8 letter-spacing**; eyebrows are uppercase tracked-out sand accent.
- **Surfaces**: warm-white cards on oat background, 8px radius, 1px subtle border, warm shadow.
- **Status meaning carried by tinted backgrounds + matched darker text** — never by colored left-border stripes (the brand explicitly retired that as "lazy").
- **Light theme only** — finishing-phase work happens on desktop in daylight studios + on phones at construction sites; dark mode is not in scope.
- **Anti-references**: cold-blue SaaS (Linear, Jira), glossy gradients (Notion AI), neon accents (Vercel), dashboard-stripe patterns (typical admin templates).

### Design Principles
1. **Truth on the page** — every record is a real DB row with a source, an author, a timestamp; the UI never invents emphasis the data doesn't support.
2. **Status as meaning, not as decoration** — the 5-step flag ladder (OK / INFO / WARNING / HIGH / CRITICAL) fires only when something needs attention; chips and tints, never decorative color.
3. **Bahasa-first** — every label, placeholder, button, empty-state, and error message is in Bahasa Indonesia by default; English exists only as a fallback for shared technical terms.
4. **The page is calm; emphasis is earned** — hierarchy through weight + spacing + a single sand accent. No multi-color rainbows, no large icons above headings.
5. **Tables and lists over cards-of-cards** — the brand's signature is the solid near-black table header row, not a nested-card maze. Don't wrap everything in cards.
6. **Document rigor** — sections feel like a spec sheet: numbered, ruled, footed with the wordmark. Generous whitespace, single-column mobile, content max-widths on desktop.

### Known Brand vs Impeccable Tensions (deliberate)
- **Space Grotesk is in impeccable's reflex_fonts_to_reject list.** The SANO brand mandates it as the single typeface. The brand wins — auditors should NOT flag Space Grotesk as an AI tell here.
- **Light theme only** is deliberate; impeccable advises against defaulting to either theme "to be safe," but the field-site daylight context + brand's "documentary/paper" mood make light the correct choice, not a default.
